INSTRUCTIONS TO RUN DeruRPY:

	1- Run [ $rosrun rosserial_python serial_node.py tcp ] in terminal to open port 11411 (only if you're using quadex).
	
	2- Run input executable from within gumstix if you want to collect data from quad (copy the "m" folder that is inside of the "quadex" folder into your gumstix) 
	   OR run ./compassInput in this folder to give it input manually.
	   
	3- Run ./compassOutput
	
	4- Data should be collected.
